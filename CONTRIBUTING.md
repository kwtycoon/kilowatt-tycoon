# Contributing

Everyone is welcome! This is a fun project, so no pressure—just enjoy building together.

## Getting Started

Check the [README](README.md) for setup instructions. Browse open issues to find something interesting, or just start hacking on whatever you want.

## Workflow

- **Share what you're working on** — let others know so we don't duplicate effort
- **Open PRs early and often** — draft PRs are great for getting feedback
- **Use AI or whatever tools you like** — totally fine, use what works for you
- **Commit freely** — small commits, big commits, whatever feels right

Have fun!
